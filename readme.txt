[center][size=16pt][b]XFire Profile Modification[/b][/size][/center]

[center][size=12pt][u][b]Compatibility[/b][/u][/size][/center]

The "[i]XFire Profile Modification[/i]" is compatible for both SMF [b]1.1.X[/b] and [b]2.0[/b]. It will be updated when any new versions of SMF are released.

[center][size=12pt][u][b]Features[/b][/u][/size][/center]

This modification allows each user to insert their XFire Username in the "Forum Profile" section of their Profile Modification, which will then come up with a small XFire Logo Icon in three places:

 � Profile Page
 � Memberlist
 � Post Profile

[center][size=12pt][u][b]Installation[/b][/u][/size][/center]

If you already have any other modifications installed on your website that modifies the MemberList, this modification may not work, however it has been tested with a variety of other modifications and works perfectly. The modification has also only been tested on the default theme of SMF; "Core". It should work on all other themes, however hasn't been tested on any.

[b]SMF 1.1.X: [/b]Installing this modification for SMF 1.1.X will only edit the template files. For all themes other than the default, just copy the "xfire.gif" file to "/Themes/YourTheme/images" folder.

[b]SMF 2.0: [/b]It should install smoothly for SMF 2.0, however you will have to check the checkboxes for each other theme that you want it installed for; "Install In Other Themes" Option when installing the package.
